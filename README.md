# HaramBot
Еще один бот для менеджмента дискорд сервера, написанный на Golang

# Сборка и запуск
1. Скопировать содержимое файла `.env.example` в `.env` и отредактировать значения внутри.
2. Собрать и запустить бота
```bash
go build && ./haram_bot 
```
